# Pipeline Run

How to start, resume, and monitor a Dexter run. For what each agent and tool does step by step ‚Äî with a concrete dishwasher example ‚Äî see [Sample Run: Dishwasher](/sample-runs/dishwasher-example).

## Start a run

### CLI (headless)

```bash
opencode run --agent orchestrator -- "build the dishwasher from input_images/dishwasher.png"
```

### TUI (interactive)

```bash
cd /path/to/dexter
opencode
```

Press **Tab** or type `/agents` and select **orchestrator**, then describe your task:

```
build the dishwasher from input_images/dishwasher.png
```

Useful slash commands: `/agents` (switch agent), `/connect` (re-authenticate), `/init` (setup), `/new` (new session).

## Resume or iterate

```bash
# Resume interrupted run
opencode run --agent orchestrator -- "resume .intermediate/dishwasher/001/"

# Continue after human parts gate
opencode run --agent orchestrator -- "parts confirmed, continue .intermediate/dishwasher/001/"

# Request more placement iterations
opencode run --agent orchestrator -- "run 3 more placement iterations for .intermediate/dishwasher/001/"
```

The orchestrator probes `.intermediate/<asset>/<NNN>/` before every step and skips outputs that already exist and validate.

## Human gates

| Gate | When | What you review |
|------|------|-----------------|
| **Parts** | After `parts.json` | Part names, descriptions, joint types, placement numbers |
| **Placement** | After the critic loop | Best iteration renders, critic score, `assembled.blend` |

## Loop exit

From `configs/base.yaml` `loop.*`: stop when `score >= score_threshold` and `N >= min_loops`, when `N >= max_loops`, or when the score has not improved for `no_improvement_patience` consecutive iterations. The best-scoring iteration is kept even if a later one regresses.

## Output directory

Every run writes to `.intermediate/<asset>/<NNN>/` (gitignored). This tree is the source of truth for run state.

‚îú‚îÄ‚îÄ iterations/

| Pattern | Example | Meaning |
|---------|---------|---------|
| `<asset>` | `dishwasher` | From `parts.json` ‚Üí `object` |

| Iteration dirs | `001` ‚Ä¶ `006` | One placement ‚Üî critic round each |

## Monitor progress

```bash
RUN=.intermediate/dishwasher/001

ls $RUN/parts.json $RUN/component_glbs/ $RUN/placement_init.json
ls $RUN/iterations/

cat $RUN/iterations/005/critic.json | python3 -c \
  "import json,sys; d=json.load(sys.stdin); print(f'Iter {d[\"iteration\"]}: {d[\"score\"]} ‚Äî {d[\"summary\"]}')"

ls $RUN/robot.usda 2>/dev/null || echo "USD not yet exported"
```

## Validate artifacts

```bash
python3 tool_scripts/common.py \
  --schema schemas/parts.schema.json \
  --data .intermediate/dishwasher/001/parts.json

python3 tool_scripts/common.py \
  --schema schemas/assembly.schema.json \
  --data .intermediate/dishwasher/001/iterations/001/assembly.json
```
